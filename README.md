# Tailscale Setup
Instructions on how to install and run Tailscale for remote Jellyfin access

## Download Tailscale
1. Navigate [here](https://tailscale.com/download/windows), select Windows
2. When it's finished downloading, run the installer (tailscale-setup-1.9.xx.exe)

## Configuration
1. Agree to the license terms and conditions and hit "Install"
2. After it's done installing, a Tailscale icon should appear in your system tray
3. Right click the icon and select "Log in"
4. A browser will open. Enter "kajlutap@gmail.com" for the email. Text me for the password

After logging in, it should say that you're connecting your device to my tailnet. Hit OK.
